Handles and processes commands as a middle-man interface to call other functions.

Parameters
commandThe command that is being executed.
sThe string used for the search_db() function in Database in order to search via the various course attributes (e.g. Humanities, Western, etc.).
courseThe name of a course's major for use with instantiating Course objects (e.g. ECE, MATH, etc.).
class_numThe course number of a course for use with instantiating Course objects (e.g. 391, 225, etc.).
gradeThe grade received for a course (e.g. B-, IP (in progress), PS (pass), NT (not taken), A-, C+, etc.).
sgroupThe string used for selecting a group when trying to find courses that belong to a certain group (e.g. CompE core courses, Engineering general education courses, Composition I courses, etc.).
genedThe string used for selecting courses that belong to a general education course group (e.g. Humanities, Western, U.S. Minority, etc.).
semesterThe string used for indicating the user's selection of a specific semester for usage with various functions, such as ones that add/remove courses from the chosen semester.
noteThe string used to store any notes that the user may have added to a semester.
